    public constructor(callback: function) source

    Params:

    NameTypeAttributeDescription
    callback function

    The callback function that forwards progress reports from the tabulation functions.

    The callback function is called with one parameter, a plain old data object with properties:

    • progressFraction = A number between 0 and 100 inclusive, indicating the current percentage of work completed
    • description = A string that gives a description of the progress that is being made.
