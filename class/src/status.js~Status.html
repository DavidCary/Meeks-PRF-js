    votes K.Decimal | null
    • optional
    • default: K.ZERO

    The number of votes for the candidate; null before the first round and for excluded candidates.
    nbrRound number
    • optional
    • default: 0

    The 1-based number of the round for which this status applies or the round in which the candidate was elected or defeated.
    status string
    • optional
    • default: K.STATUS.hopeful

    A string indicating the status of the candidate. The value should be one in K.STATUS.
    keepFactor number
    • optional
    • default: K.ONE

    The fraction of a ballot's vote weight that is counted for the candidate.
    destiny string
    • optional
    • default: K.DESTINY.normal

    An indication of whether the candidate was pre-designated as excluded or protected. The value should be one in K.DESTINY.
